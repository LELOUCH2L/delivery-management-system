# Delivery Management System

## Project Overview
This project implements a **Delivery Management System** in Python, demonstrating **object-oriented programming (OOP)** approaches.

---

## Features
- Add and manage customers and drivers
- Customers: Place order
- Drivers: Deliver order
- Delivery Order: Assign driver

---

## How to run
1. Open a terminal in the project folder.
2. Run tests in your terminal by:
```bash
python delivery_management_system/main.py
```
